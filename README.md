# AC SHOP
### 一個簡易購物的網頁
<br>

## 功能
<br>

* 可以選擇 ＋ - 變更購物數量以及金額
* 切換按鈕，可以掌控整體進度
* ＲＷＤ設計：隨著畫面改變手機版，桌機版網頁

## 下載方式
<br>
  git clone https://github.com/MikeLee0358/SHOP.git

  
cd Shop

npm install

vs code底下對index.html點選右鍵 開啟Open with Live Server


</br>
